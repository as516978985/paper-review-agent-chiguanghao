# 参考文献审核

## 执行流程

1. 读取 `references/review-rules.md`。
2. 从文档中识别完整参考文献列表，保留每条原文。
3. 存在 DOI 时调用 `check_citation`，同时传入 `doi`、`title`、`author`。
4. 不存在 DOI 时调用 `search_by_title`，同时传入 `title` 和 `author`（作者名能显著提高匹配准确率）。
5. 记录数据源、登记题名、作者、年份和字段差异。
6. 调用 `run_recent_ratio` 计算近五年数量与占比。
7. 汇总格式问题、文献证据、优先修改项和总体结论。
8. 存在历史记录时，对比已修正、仍存在、新增和无法判断。

## 作者匹配说明

- 从参考文献原文中提取第一作者姓氏（如 "Brown T B" → "Brown"）
- 将作者名传入 `search_by_title` 或 `check_citation` 的 `author` 参数
- 系统会综合题名相似度（权重 0.6）和作者匹配度（权重 0.4）排序
- 综合得分 >= 0.75 时判定为已核实

## 证据约束

- 不把模型回忆写成外部查询结果。
- 查询无结果时写"未能核实"。
- 网络失败时写"暂缓判断"。
- 历史意见不能替代本次重新检查。

## 输出结构

1. 基本信息
2. 格式检查摘要
3. 格式问题明细
4. 参考文献逐条核验
5. 近五年统计
6. 优先修改项
7. 与上次审核对比
8. 总体结论
9. 限制说明
